## ✨ Features

### Core Functionality
- ✅ **Add Tasks** - Quickly add new tasks with a clean input interface
- ✅ **Mark Complete** - Click checkboxes to toggle task completion status
- ✅ **Delete Tasks** - Remove individual tasks with a single click
- ✅ **Clear Completed** - Bulk remove all completed tasks at once
- ✅ **Filter Tasks** - View All, Active, or Completed tasks separately

### Design & UX
- 🎨 **Neumorphic Design** - Soft, tactile UI with depth and elegance
- 📱 **Fully Responsive** - Perfect on desktop, tablet, and mobile devices
- ⚡ **Smooth Animations** - Fluid transitions and delightful interactions
- 🎯 **Beautiful Gradients** - Purple gradient accents throughout
- 🌈 **Soft Color Palette** - Easy on the eyes with harmonious colors

### Data Management
- 💾 **Local Storage** - All tasks automatically saved to browser storage
- 🔄 **Persistent Data** - Tasks remain after closing the browser
- 📊 **Real-time Statistics** - Live counters for Total, Active, and Completed tasks
- 🕐 **Timestamps** - View when each task was created

### Additional Features
- 🔤 **Character Counter** - Shows text length (max 50 characters)
- ⌨️ **Keyboard Support** - Press Enter to add tasks
- 🚫 **Input Validation** - Shake animation for empty submissions
- ♿ **Accessible** - Clean semantic HTML structure
- 🎭 **Customizable** - Easy to modify colors and styles

## 🚀 Getting Started

### Quick Setup

1. **Download or Clone**
   ```bash
   git clone https://github.com/RohithAnupoju18/neumorphic-todo.git
   cd neumorphic-todo
   ```

2. **Install Dependencies (if any)**
   ```bash
   npm install
   ```

3. **Run Locally**
   Open `App.html` in your browser or use a dev server such as `live-server` or `http-server`.
